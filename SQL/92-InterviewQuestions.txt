1. SQL Query to get Nth Highest Salary 
-> SELECT salary FROM employees ORDER BY salary DESC LIMIT N-1, 1;
-> SELECT salary FROM employees ORDER BY salary DESC LIMIT 1, OFFSET N-1;
-> SELECT salary FROM employees e1 
    WHERE N - 1 = (SELECT COUNT(DISTINCT salary) FROM employees e2 WHERE e2.salary > e1.salary);


2. SQL Query to get 2nd Highest Salary
-> SELECT MAX(salary) FROM employees NOT IN (SELECT MAX(salary) FROM employees);
-> SELECT MAX(salary) FROM employees <> (SELECT MAX(salary) FROM employees);


3. SQL Query to get Employee name start with 'A'
-> SELECT * FROM employees WHERE eName LIKE 'A%';

6. SQL Query to fetch alternate records from a table
-> SELECT * FROM table_name WHERE column_name % 2 = 0; (Even Records)
-> SELECT * FROM table_name WHERE column_name % 2 = 1; (Odd Records)
